# Карта файлов демо QNP Tender+

| Файл | Что это |
|------|---------|
| `index.html` | Главная: список тендеров |
| `learn.html` | Обучение: чат по базе |
| `library.html` | Библиотека: презентации и тесты |
| `estimator.html` | ИИ-сметчик: загрузка ТЗ |
| `calc.html` | Предварительный расчёт сметы |
| `scout.html` | Разведчик закупок |
| `settings.html` | Настройки поиска |
| `tender.html` | Карточка одного тендера |
| `css/app.css` | Общие стили |
| `js/data.js` | Тексты, тендеры, курсы, ответы чата |
| `js/home.js` | Логика главной |
| `js/learn.js` | Логика чата |
| `js/library.js` | Логика презентаций/тестов |
| `js/tender.js` | Логика карточки тендера |
| `js/nav.js` | Подсветка активной кнопки |
| `scope.md` | Границы проекта |

В правом нижнем углу каждой страницы подсказка, какой файл сейчас открыт.
