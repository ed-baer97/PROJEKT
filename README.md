# QNP Tender+ — демо-прототип

Интерактивный HTML-макет платформы тендерной аналитики (стиль QNP) с модулем **обучения ПМ**: презентации, тесты и чат-бот по Word-базе.

Границы проекта (scope): [`scope.md`](scope.md).

---

## Как открыть

1. Откройте `index.html` в браузере (двойной клик или Live Server).
2. Сервер не нужен — обычные статические файлы.
3. Для GitHub Pages: ветка `main`, корень репозитория → сайт откроется с `index.html`.

В правом нижнем углу страниц — подсказка, какой файл сейчас редактируете.

---

## Карта страниц

| Страница | Файл | Что править |
|----------|------|-------------|
| **Главная** — список тендеров | `index.html` | Шапка, hero, фильтры, таблица |
| **Обучение** — презентация + тест + чат | `learn.html` | Вёрстка экрана обучения |
| **Библиотека знаний** — документы | `library.html` | Список слева, текст справа |
| **ИИ-сметчик** | `estimator.html` | Загрузка ТЗ, шаги процесса |
| **Предварительный расчёт** | `calc.html` | Параметры и результат сметы |
| **Разведчик** | `scout.html` | Поиск ГПЗ / госэкспертизы |
| **Настройки поиска** | `settings.html` | Ключевые слова, пороги |
| **Карточка тендера** | `tender.html` | Детали лота + чат разбора кейса |

### Навигация с главной

Кнопки шапки: **ИИ сметчик** · **Обучение** · **Настройки поиска** · Загрузить.

**Библиотека знаний** с главной убрана — открывается кнопкой со страницы **Обучение**.

Разведчик — кнопка «Открыть» в блоке hero.

## Модуль обучения (фаза 1)

На странице `learn.html`:

1. **Презентация** — ИИ-слайды по разделу (данные из `QNP.sections`).
2. **Тест** — вопросы по тому же разделу.
3. **Чат-бот** справа — отвечает только по базе; вне базы говорит «в базе этого нет».

Связка: изучил → прошёл тест → спросил бота.

Библиотека (`library.html`): только документация — клик по файлу показывает содержимое справа.

## Фаза 2 (в макете есть экраны)

В границах проекта — отдельный этап после обучения. В демо уже набросаны экраны:

- Разведчик (`scout.html`)
- ИИ-сметчик / Excel (`estimator.html`, `calc.html`)
- Карточка тендера (`tender.html`) — чат **«Разбор кейса»**: маржа, риски, налоги, сроки по предрасчёту + база знаний
- Аналитика закупок — под вопросом (см. `scope.md`)
